# Roadmap

## Planned

- WebSocket support via decorators
- Database adapters (Prisma, Drizzle, TypeORM)
- GraphQL module
- Rate limiting middleware
- Session management
- CLI `kick deploy` command
- Plugin system for community extensions

## Under Consideration

- gRPC support
- Queue/worker module (BullMQ, SQS)
- Multi-tenancy helpers
- OpenTelemetry integration
- SSE (Server-Sent Events) support

- v0.3.1 — Per-package README/LICENSE, npm scope rename
- v0.3.0 — Monorepo rewrite with custom DI, Express 5, Zod, Vite HMR
