# @forinda/kickjs-core

> **Note:** This package is included in [`@forinda/kickjs`](https://www.npmjs.com/package/@forinda/kickjs). For new projects, install the unified package instead:
>
> ```bash
> pnpm add @forinda/kickjs
> ```
>
> `@forinda/kickjs-core` continues to work for existing projects.

Core DI container, decorators, module system, logger, and error types for KickJS.
